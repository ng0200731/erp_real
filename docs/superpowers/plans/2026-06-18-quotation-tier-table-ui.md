# Quotation Tier Table — Phase 2 (In-App UI) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the quotation pricing mini-tab's flat/tier toggle + single template picker with a conditional `none`/`brand`/`factory` dropdown whose options are driven by the Product Details' brand + customer, scope-aware tier-table pickers that load quantity breakpoints into an editable tier grid, and persistence of the new `tierScopeMode` pricing shape for all 8 product types.

**Architecture:** The pricing mini-tab is generated by ONE function — `generateQuotationContent(tagType, tagName, source, isDummy2Mode, mode)` (~line 18991) — shared by all 3 create flows (email-receive-view at 10512, quotation-create at 17787, outsource at 17996). So one set of edits covers every flow. Wiring lives in `initPricingMode(scope)` (~19338-19467), which is container-scoped (`getActiveQuotationContainer()`). Collection lives in `collectProductDetailsFromForm(tagType)` (~19193-19243). Phase 1 already delivers the filtered `/api/pricing-tier-tables?scope=&brandId=&customerName=` endpoint and on-read migration, so the UI consumes the new shape directly.

**Tech Stack:** Vanilla JS + HTML (inline in `public/index.html`), Express backend (Phase 1 endpoints), Playwright E2E (the project's test tool — `npx playwright test`).

## Scope (this phase)

- Create-flow pricing UI only (the 3 flows above). The separate **edit/view** form (`generateQuotationContentForView`, ~18818) does NOT render a pricing mini-tab today — that pre-existing gap is out of scope; tier selection is create-time. (Noted for a future phase.)
- Conditional dropdown + scope pickers + tier grid + new-shape persistence + all 8 product types.
- No supplier email / portal changes (Phase 3).

## Global Constraints

- All edits go through the shared `generateQuotationContent` HTML string, `initPricingMode(scope)`, and `collectProductDetailsFromForm(tagType)` — container-scoped, no global-state assumptions.
- `tierScopeMode` persisted values are exactly `'none'` | `'brand'` | `'customer'`. The UI dropdown uses `'none'` | `'brand'` | `'factory'`; **`'factory'` maps to `'customer'`** on save.
- `disabled` tier tables are excluded by the Phase 1 filter endpoint — the UI must not re-filter.
- Preserve the existing tier-grid helpers (`syncBaseFromTiers`, `recalcRowTotal`, `addTierRow`) and the flat-pricing fields (`#quotationQuantity` / `#quotationUnitPrice` / `#quotationTotal`); "None" mode must behave exactly like today's flat mode.
- Pricing block must be collected for **all 8** product types (today only PU Patch + generic).
- Old controls removed: `#tierTemplateSelect`, `#applyTierTemplateBtn`, `#tierTableSelectWrap`, hidden `#selectedTierTemplateId`.
- Tests run via `npx playwright test tests/unit-pricing-*.spec.js --project=chromium` (Playwright auto-starts `npm start` on :3001; webServer env carries `PLAYWRIGHT_TEST=true`).

- [ ] **Step 4: Replace `initPricingMode` (keep the tier-grid helpers)**

Replace the **entire** `function initPricingMode(scope) { … }` (current ~19338-19467) with the version below. The `syncBaseFromTiers` / `recalcRowTotal` / `addTierRow` helpers and flat-field behavior are preserved verbatim; the mode-toggle and template-apply logic are replaced by the conditional dropdown + scope pickers.

```js
    function initPricingMode(scope) {
      const s = scope || document;
      const modeSel = s.querySelector('#pricingModeSelect');
      const flatSec = s.querySelector('#flatPricingSection');
      const tierSec = s.querySelector('#tierPricingSection');
      const rowsWrap = s.querySelector('#tierRows');
      const addRowBtn = s.querySelector('#addTierRowBtn');
      const qty = s.querySelector('#quotationQuantity');
      const unit = s.querySelector('#quotationUnitPrice');
      const tot = s.querySelector('#quotationTotal');
      const brandWrap = s.querySelector('#brandTierWrap');
      const brandSel = s.querySelector('#brandTierTableSelect');
      const custWrap = s.querySelector('#customerTierWrap');
      const custSel = s.querySelector('#customerTierTableSelect');
      if (!modeSel || !flatSec || !tierSec || !rowsWrap) return;

      function syncBaseFromTiers() {
        const first = rowsWrap.querySelector('.tier-row');
        if (!first || !qty || !unit || !tot) return;
        const q = parseInt(first.querySelector('.tier-qty')?.value || '0', 10) || 0;
        const u = parseFloat(first.querySelector('.tier-unit')?.value || '0') || 0;
        const t = q * u;
        qty.value = q > 0 ? String(q) : '';
        unit.value = u > 0 ? u.toFixed(2) : '';
        tot.value = (q > 0 && u >= 0) ? t.toFixed(2) : '';
      }

      function recalcRowTotal(row) {
        const q = parseInt(row.querySelector('.tier-qty')?.value || '0', 10) || 0;
        const u = parseFloat(row.querySelector('.tier-unit')?.value || '0') || 0;
        row.querySelector('.tier-total').value = (q * u).toFixed(2);
        syncBaseFromTiers();
      }

      function addTierRow(initQ = '', initU = '') {
        const row = document.createElement('div');
        row.className = 'tier-row';
        row.style.cssText = 'display:grid; grid-template-columns:1fr 1fr 1fr 80px; gap:8px; margin-bottom:6px;';
        row.innerHTML =
          '<input type="number" class="tier-qty" min="1" placeholder="Qty" style="padding:4px; border:1px solid #ccc;">' +
          '<input type="number" class="tier-unit" min="0" step="0.01" placeholder="Unit Price" style="padding:4px; border:1px solid #ccc;">' +
          '<input type="text" class="tier-total" readonly placeholder="Total" style="padding:4px; border:1px solid #ccc; background:#f5f5f5;">' +
          '<button type="button" class="btn remove-tier" style="padding:4px 8px;">Remove</button>';
        rowsWrap.appendChild(row);
        const qEl = row.querySelector('.tier-qty');
        const uEl = row.querySelector('.tier-unit');
        const rm = row.querySelector('.remove-tier');
        if (initQ !== '') qEl.value = initQ;
        if (initU !== '') uEl.value = parseFloat(initU).toFixed(2);
        qEl.addEventListener('input', () => recalcRowTotal(row));
        uEl.addEventListener('input', () => recalcRowTotal(row));
        rm.onclick = () => { row.remove(); syncBaseFromTiers(); };
        recalcRowTotal(row);
      }

      if (addRowBtn) {
        addRowBtn.onclick = () => addTierRow();
      }

      // ---- field readers (container-scoped) ----
      function brandIdEl() { return s.querySelector('#quotationBrandId') || document.getElementById('quotationBrandId'); }
      function custNameEl() { return s.querySelector('#quotationCustomerName') || document.getElementById('quotationCustomerName'); }
      function brandIdVal() { const el = brandIdEl(); return el ? String(el.value || '').trim() : ''; }
      function customerNameVal() { const el = custNameEl(); return el ? String(el.value || '').trim() : ''; }

      // ---- conditional pricing-mode dropdown ----
      function populateModeOptions() {
        const opts = ['<option value="none">None (flat)</option>'];
        if (brandIdVal()) opts.push('<option value="brand">Brand</option>');
        if (customerNameVal()) opts.push('<option value="factory">Garment Factory</option>');
        const prev = modeSel.value;
        modeSel.innerHTML = opts.join('');
        modeSel.value = Array.from(modeSel.options).some((o) => o.value === prev) ? prev : 'none';
      }

      // ---- load one tier table's breakpoints into the grid ----
      async function loadTierTableIntoGrid(tableId) {
        rowsWrap.innerHTML = '';
        if (!tableId) return;
        try {
          const res = await fetch('/api/pricing-tier-tables/' + encodeURIComponent(tableId));
          const data = await res.json();
          const tiers = data && data.success && data.table ? (data.table.tiers || []) : [];
          tiers.forEach((t) => addTierRow(t.quantity || 0, t.unitPrice || 0));
        } catch (e) {
          console.error('Failed to load tier table', e);
        }
      }

      // ---- populate a scope picker from the filtered API (auto-show if exactly one) ----
      async function populateScopePicker(selEl, url) {
        if (!selEl) return;
        let items = [];
        try {
          const res = await fetch(url);
          const data = await res.json();
          items = data && data.success ? (data.tables || []) : [];
        } catch (e) {
          console.warn('Failed to load tier tables', e);
        }
        if (items.length === 0) {
          selEl.innerHTML = '<option value="">(no tier table — create one)</option>';
          return;
        }
        selEl.innerHTML = ['<option value="">Select…</option>']
          .concat(items.map((p) => `<option value="${p.id}">${escapeHtml(p.name || 'Untitled')}</option>`))
          .join('');
        if (items.length === 1) {
          selEl.value = String(items[0].id);
          await loadTierTableIntoGrid(items[0].id);
        }
      }

      async function refreshBrandPicker() {
        const bid = brandIdVal();
        if (bid) await populateScopePicker(brandSel, '/api/pricing-tier-tables?scope=brand&brandId=' + encodeURIComponent(bid));
      }
      async function refreshCustomerPicker() {
        const cn = customerNameVal();
        if (cn) await populateScopePicker(custSel, '/api/pricing-tier-tables?scope=customer&customerName=' + encodeURIComponent(cn));
      }

      // ---- mode change drives which picker + grid show ----
      async function onModeChange() {
        const mode = modeSel.value || 'none';
        if (brandWrap) brandWrap.style.display = 'none';
        if (custWrap) custWrap.style.display = 'none';
        if (mode === 'none') {
          flatSec.style.display = 'grid';
          tierSec.style.display = 'none';
          rowsWrap.innerHTML = '';
          return;
        }
        flatSec.style.display = 'none';
        tierSec.style.display = 'block';
        rowsWrap.innerHTML = '';
        if (mode === 'brand') {
          if (brandWrap) brandWrap.style.display = '';
          await refreshBrandPicker();
        } else if (mode === 'factory') {
          if (custWrap) custWrap.style.display = '';
          await refreshCustomerPicker();
        }
        if (!rowsWrap.querySelector('.tier-row')) addTierRow(qty ? qty.value : '', unit ? unit.value : '');
        syncBaseFromTiers();
      }
      modeSel.onchange = onModeChange;
      modeSel.addEventListener('focus', populateModeOptions);
      if (brandSel) brandSel.onchange = () => loadTierTableIntoGrid(brandSel.value);
      if (custSel) custSel.onchange = () => loadTierTableIntoGrid(custSel.value);

      // ---- re-evaluate when customer/brand change ----
      function onCustomerOrBrandChange() {
        const before = modeSel.value;
        populateModeOptions();
        if (before === 'brand' && !brandIdVal()) { modeSel.value = 'none'; return onModeChange(); }
        if (before === 'factory' && !customerNameVal()) { modeSel.value = 'none'; return onModeChange(); }
        if (modeSel.value === 'brand') refreshBrandPicker();
        else if (modeSel.value === 'factory') refreshCustomerPicker();
      }
      const bEl = brandIdEl();
      const cEl = custNameEl();
      if (bEl) bEl.addEventListener('change', onCustomerOrBrandChange);
      if (cEl) {
        cEl.addEventListener('change', onCustomerOrBrandChange);
        cEl.addEventListener('input', onCustomerOrBrandChange);
      }

      populateModeOptions();
      onModeChange();
    }
```
