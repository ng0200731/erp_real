Status Breakdown Popup — Approaches

How should the drill-down popup work when you click a status?

A

Full-Screen Overlay Table

Click a status → full-screen modal overlay with a data table. Export button in the top-right corner. Clean, focused view. Easy to scan many rows.

B

Side Panel Slide-In

Click a status → a panel slides in from the right side, showing the table alongside the expanded chart. Keeps context visible but less space for data.

C

Centered Modal Dialog

Click a status → a centered modal dialog (~80% viewport) with scrollable table. Export button at bottom. Classic modal pattern. Balanced context.

Recommendation

I recommend A — Full-Screen Overlay because:

  • Most data space for the table — fits all columns comfortably
  • Export button is prominent and easy to find
  • Matches the monochrome, focused design of the existing dashboard
  • Same pattern as the existing chart expand behavior
